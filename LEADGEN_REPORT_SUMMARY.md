## 🔍 Gap Analysis Checklist

When reviewing each prospect's website, check:

| # | Criteria | Hot Lead Trigger |
|---|----------|------------------|
| 1 | **Website exists?** | ❌ No website = 🔥 HOT |
| 2 | Mobile responsive? | ❌ Not mobile = Warm |
| 3 | Load speed | >3 seconds = problem |
| 4 | SSL Certificate | HTTP not HTTPS = gap |
| 5 | Contact forms | Broken/missing = gap |
| 6 | E-commerce | Can't buy online = gap |
| 7 | SEO basics | Poor meta = gap |
| 8 | Social proof | No testimonials = gap |
| 9 | Blog/Content | No updates since 2022 = gap |
| 10 | CRM integration | No chat widget = gap |

**Scoring:** 5+ gaps = HIGH priority | 3-4 gaps = MEDIUM | 1-2 gaps = LOW
