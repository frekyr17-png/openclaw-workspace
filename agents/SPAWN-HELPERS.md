# Quick Agent Spawning

## Talk to Richie (Trading)
```bash
openclaw run --label richie-chat "Richie, [your message]"
```

## Talk to Aria (Project Management)
```bash
openclaw run --label aria-chat "Aria, [your message]"
```

## Talk to Mr Bean (Security)
```bash
openclaw run --label mrbean-chat "Mr Bean, [your message]"
```

## From Telegram
Use /spawn:
- `/spawn richie "your message"`
- `/spawn aria "your message"`  
- `/spawn mrbean "your message"`

Each agent runs, responds, then exits.
